# GoblinFTP Phase 1 — Scaffold Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create the full repository skeleton — working Go backend (Echo + `/healthz`), buildable Nuxt 3 SPA, multi-stage Dockerfile, justfile, docker-compose, and CI — all passing before any feature code is written.

**Architecture:** Go binary (Echo) and Nuxt 3 static SPA live in the same Docker image. Caddy serves static files from `/app/public/` and reverse-proxies `/api/*` to the Go binary on `:8080`. The backend exposes only `/healthz` at this phase; all feature routes are added in later phases.

**Tech Stack:** Go 1.22+, Echo v4, github.com/stretchr/testify, Nuxt 3 (ssr:false), @nuxt/ui v3, @pinia/nuxt, @nuxtjs/i18n v9, Caddy 2, Docker, GitHub Actions, `just` (task runner)

```go
package main

import (
	"net/http"
	"os"

	"github.com/labstack/echo/v4"
	"github.com/labstack/echo/v4/middleware"
)

func newApp() *echo.Echo {
	e := echo.New()
	e.HideBanner = true
	e.Use(middleware.Recover())
	e.Use(middleware.Logger())

	e.GET("/healthz", func(c echo.Context) error {
		return c.JSON(http.StatusOK, map[string]string{"status": "ok"})
	})

	return e
}

func main() {
	e := newApp()
	port := os.Getenv("GFTP_PORT")
	if port == "" {
		port = "8080"
	}
	e.Logger.Fatal(e.Start(":" + port))
}
```

- [ ] **Step 1: Create Dockerfile**

Create `Dockerfile` at repo root:

```dockerfile
# syntax=docker/dockerfile:1

# ── Stage 1: Build Nuxt SPA ────────────────────────────────────────────────────
FROM node:20-alpine AS frontend-builder
WORKDIR /build/frontend
COPY frontend/package*.json ./
RUN npm ci --prefer-offline
COPY frontend/ ./
RUN npm run generate

# ── Stage 2: Build Go binary ───────────────────────────────────────────────────
FROM golang:1.22-alpine AS backend-builder
WORKDIR /build/backend
COPY backend/go.mod backend/go.sum ./
RUN go mod download
COPY backend/ ./
RUN CGO_ENABLED=0 GOOS=linux go build -ldflags="-w -s" -o /app/gftp ./cmd/gftp

# ── Stage 3: Final image ───────────────────────────────────────────────────────
FROM caddy:2-alpine

RUN mkdir -p /app/public /app/data /app/storage

COPY --from=frontend-builder /build/frontend/.output/public /app/public
COPY --from=backend-builder /app/gftp /app/gftp
COPY docker/Caddyfile /etc/caddy/Caddyfile
COPY docker/entrypoint.sh /entrypoint.sh

RUN chmod +x /entrypoint.sh

EXPOSE 80
ENTRYPOINT ["/entrypoint.sh"]
```

- [ ] **Step 3: Create docker/Caddyfile**

Create `docker/Caddyfile`:

```
:80 {
    # API requests → Go binary
    handle /api/* {
        reverse_proxy localhost:8080
    }

    # SSO entry point → Go binary (must intercept before SPA fallback)
    handle {
        uri has_query sso
        reverse_proxy localhost:8080
    }

    # All other requests → Nuxt SPA with HTML5 history fallback
    handle {
        root * /app/public
        try_files {path} /index.html
        file_server
    }
}
```

- [ ] **Step 4: Create docker/entrypoint.sh**

Create `docker/entrypoint.sh`:

```bash
#!/bin/sh
set -e

# Start Go backend in background
/app/gftp &

# Wait for backend to be ready (up to 5 seconds)
for i in $(seq 1 10); do
    if wget -q -O /dev/null http://localhost:8080/healthz 2>/dev/null; then
        break
    fi
    sleep 0.5
done

# Start Caddy in foreground (PID 1)
exec caddy run --config /etc/caddy/Caddyfile --adapter caddyfile
```
